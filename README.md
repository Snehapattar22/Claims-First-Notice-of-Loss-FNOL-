# Claims First Notice of Loss FNOL(LLM-Powered Insurance Claim Intake System)
## Overview
### FNOL Claims Assistant is an AI-powered web application that simplifies the First Notice of Loss (FNOL) process for insurance claims. It uses a Large Language Model (LLM) to guide users empathetically, collect claim details, extract information from uploaded documents, and submit claims efficiently.

## Key Features
- Empathetic AI Assistant for stress-free claim guidance
- Structured Claim Data Collection
- Document Upload & Text Extraction (FIR / accident reports)
- LLM-based Information Extraction
- Automatic Claim ID Generation
- SQLite Database Storage

## Tech Stack
- Backend: Python, Flask
- LLM: Google Gemini API
- Database: SQLite
- Frontend: HTML, CSS

## Application Flow
1. User describes the accident in natural language
2. AI assistant responds empathetically and guides next steps
3. User fills claim form and uploads documents
4. System extracts and stores structured claim data
5. Claim is submitted successfully with a unique Claim ID

## How to run this project
- pip install -r requirements.txt
- python app.py
